# Actix Web Examples

This folder contain just a few standalone code samples. There is a much larger registry of example projects [in the examples repo](https://github.com/actix/examples).

## Available Examples

- **dependency_injection.rs** - A complete REST API demonstrating dependency injection with fundle, error handling with ohno, and database access with sea-orm. Supports both SQLite and PostgreSQL. [See detailed README](dependency_injection_README.md)
